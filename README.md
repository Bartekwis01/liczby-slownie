# liczby-slownie
A program that converts numbers into words in polish. E.g 123 -- > sto dwadziescia trzy

---

## Limitations
- This program operates on numbers between -999 999 999 and +999 999 999
- This program doesn't use polish characters such as ฤ and รณ

---

### Expanding
- This program should be relativly easy to expand by just duplicating the millions code and replacing each mention of million with billion etc.
